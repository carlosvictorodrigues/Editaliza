Criar Favicon para Editaliza

Esta ferramenta cria favicons PNG com a letra "E" da Editaliza usando as cores da marca.

16x16 pixels

32x32 pixels

48x48 pixels

Como usar:

  1. Clique nos botões de download para baixar cada tamanho
  2. Salve os arquivos na pasta public/ do projeto
  3. O arquivo principal deve ser salvo como favicon.png
  4. Os outros tamanhos podem ser referenciados no HTML se necessário

Cores usadas:

  • Fundo: #0528f2 (azul da marca Editaliza)
  • Texto: Branco (#ffffff)
  • Fonte: Arial Bold
